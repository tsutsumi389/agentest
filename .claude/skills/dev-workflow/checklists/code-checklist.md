# Code Checklist

実装完了前の確認事項。

## 機能

- [ ] 実装プランの要件を満たしている
- [ ] エッジケースが考慮されている
- [ ] エラーハンドリングが適切

## コード品質

- [ ] TypeScript strict modeでエラーなし
- [ ] ESLint警告・エラーなし
- [ ] 命名が明確で一貫している
- [ ] 関数が単一責任
- [ ] 不要なコメント・デッドコードがない

## 規約準拠

- [ ] コーディング規約に従っている
- [ ] ファイル配置が正しい
- [ ] Import順序が正しい

## セキュリティ

- [ ] 入力バリデーションが実装されている
- [ ] SQLインジェクション対策
- [ ] 認証・認可が適切
- [ ] センシティブ情報がログ出力されていない

## パフォーマンス

- [ ] N+1クエリがない
- [ ] 適切なインデックスが設定されている
- [ ] 不要なデータ取得がない

## テスタビリティ

- [ ] 依存関係が注入可能
- [ ] モック可能な設計

## ビルド・動作確認

- [ ] `npm run build` 成功
- [ ] `npm run typecheck` 成功
- [ ] `npm run lint` 成功
- [ ] 手動での基本動作確認済み

## Git

- [ ] コミットメッセージがConventional Commits形式
- [ ] 適切な粒度でコミットされている
- [ ] 不要なファイルがコミットされていない

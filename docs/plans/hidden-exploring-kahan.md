# プラン: システム管理者機能要件ドキュメント作成

## 目的

SaaSプラットフォームの「システム管理者」向け機能要件を `/docs/requirements/admin-system.md` に出力する。

---

## 出力先

```
/docs/requirements/admin-system.md
```

---

## ドキュメント構成

既存の `user-organization.md` 形式に合わせて以下の構成で作成:

1. **概要** - システム管理者機能の目的と対象
2. **管理者ロール定義** - SUPER_ADMIN / ADMIN / VIEWER
3. **MVP機能（初期リリース）** - ID付きテーブル形式
4. **Phase 2 機能（将来）** - ID付きテーブル形式
5. **権限マトリクス** - ロール別操作可否
6. **監査ログ対象イベント** - 管理者操作のログ
7. **非機能要件** - セキュリティ、認証
8. **データベース設計への影響** - 新規テーブル
9. **関連ドキュメント**

---

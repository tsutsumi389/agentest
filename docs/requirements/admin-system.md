# システム管理者機能

## 概要

Agentest の SaaS プラットフォーム全体を管理するシステム管理者向けの機能要件。ユーザー・組織の管理、課金管理、利用状況モニタリング、システム監視、監査・コンプライアンス対応を行う。

---

## 管理者ロール定義

| ロール | 説明 | 権限 |
|--------|------|------|
| SUPER_ADMIN | 最高権限管理者 | 全機能へのアクセス + 管理者アカウント管理 |
| ADMIN | 一般管理者 | 日常運用に必要な管理機能へのアクセス |
| VIEWER | 閲覧専用 | ダッシュボード・ログの閲覧のみ |

## 監査ログ対象イベント

### 管理者操作

| カテゴリ | イベント |
|----------|----------|
| 管理者認証 | 管理画面ログイン、ログアウト、2FA認証 |
| ユーザー管理 | ユーザー停止、有効化、削除 |
| 組織管理 | 組織停止、有効化、削除 |
| 課金管理 | プラン変更、支払いステータス更新、返金処理 |
| お知らせ管理 | お知らせ作成、編集、削除、公開 |
| 管理者管理 | 管理者アカウント作成、ロール変更、削除 |
| 設定変更 | 機能フラグ変更、プラン制限値変更 |

### セキュリティイベント

| カテゴリ | イベント |
|----------|----------|
| 認証失敗 | ログイン失敗、2FA失敗 |
| 不正アクセス | 権限外の操作試行、許可外IPからのアクセス |
| セッション | 強制ログアウト、セッション失効 |

## 非機能要件

### セキュリティ

| 項目 | 要件 |
|------|------|
| 認証方式 | 管理者専用アカウント + 2FA必須（TOTP） |
| セッション有効期限 | 2時間（アクティビティに応じて延長、最大8時間） |
| パスワードポリシー | 12文字以上、大小英字・数字・記号を含む |
| ログイン試行制限 | 5回失敗で30分ロック |
| IPホワイトリスト | 管理画面は許可IPからのみアクセス可（Phase2） |

### 監査・コンプライアンス

| 項目 | 要件 |
|------|------|
| 管理者操作ログ保持 | 無期限（法的要件に応じて） |
| ログ改ざん防止 | 追記のみ、削除・編集不可 |
| エクスポート形式 | CSV、JSON |

## データベース設計への影響

### 新規追加が必要なテーブル

| テーブル | 用途 |
|---------|------|
| `AdminUser` | システム管理者アカウント |
| `AdminRole` | 管理者ロール定義 |
| `AdminSession` | 管理者セッション |
| `AdminAuditLog` | 管理者操作監査ログ |
| `SystemAnnouncement` | システムお知らせ |
| `MaintenanceNotice` | メンテナンス通知 |
| `FeatureFlag` | 機能フラグ（Phase2） |

### 主要テーブルの関係

```
AdminUser
├── AdminRole (N:1)
├── AdminSession (1:N)
└── AdminAuditLog (1:N)

SystemAnnouncement
└── AdminUser (作成者、N:1)

MaintenanceNotice
└── AdminUser (作成者、N:1)
```

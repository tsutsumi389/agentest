# ユーザー・オーガナイゼーション機能

## 概要

Agentest のユーザー管理、組織管理、料金体系、権限管理の機能要件。GitHub ライクな個人/組織の階層構造と料金体系を採用。

---

## 料金プラン

### 個人プラン

| プラン | 料金 | 対象 |
|--------|------|------|
| Free | $0/月 | 個人利用・評価目的 |
| Pro | $10/月 | 個人の本格利用 |

### 組織プラン

| プラン | 料金 | 対象 |
|--------|------|------|
| None | - | 組織作成直後のデフォルト（プロジェクト作成不可） |
| Team | $6/ユーザー/月（最低10ユーザー） | 小〜中規模チーム |
| Enterprise | 要問合せ | 大規模組織・高度なセキュリティ要件 |

### プラン別機能比較

**データ作成は全プラン無制限**（プロジェクト数、テストスイート数、テストケース数に制限なし）

| 機能 | Free | Pro | Team | Enterprise |
|------|:----:|:---:|:----:|:----------:|
| プロジェクト数 | 無制限 | 無制限 | 無制限 | 無制限 |
| テストケース数 | 無制限 | 無制限 | 無制限 | 無制限 |
| MCP セッション | 100/月 | 1,000/月 | 5,000/月 × ユーザー数（組織プール） | 無制限 |
| MCP 同時接続数 | 1 | 3 | 5 × ユーザー数（組織プール） | 無制限 |
| 実行履歴保持期間 | 90日* | 1年* | 1年* | 無制限 |
| エビデンスストレージ | 無制限** | 無制限** | 無制限** | 無制限 |
| 組織メンバー数 | - | - | 無制限 | 無制限 |
| SSO (SAML/OIDC) | - | - | - | ○ |
| 監査ログ | - | - | 90日 | 無制限 |
| 専用サポート | - | - | - | ○ |
| SLA保証 | - | - | 99.9% | 99.99% |

*各テストケースの最新実行結果は期間経過後も保持
**エビデンスは実行履歴と連動して削除（履歴保持期間を超えたものは自動削除、ただし最新結果分は保持）

### 使用量制限の考え方

- **MCP セッション**: エージェントの一連の操作を1セッションとしてカウント（月次リセット）
  - 例：テストケース作成依頼 → テストスイート作成 → テストケース作成の繰り返し → エージェント終了 = 1セッション
  - 個々のツール呼び出しではなく、エージェントが停止するまでの一連の流れで1回
  - Team プランでは組織全体でセッション数をプール（例：10ユーザーなら月間50,000セッション）
- **MCP 同時接続数**: Coding Agent が MCP サーバーに接続する数
  - 複数の Coding Agent を並列実行してテスト作成・実行を同時に行う場合に使用
  - Team プランでは組織全体でプール（例：10ユーザーなら同時50接続まで）
- **エビデンス**: スクリーンショット等は1ファイル10MBまで、実行履歴削除時に連動削除
- **実行履歴保持**: 期間経過後も各テストケースの最新実行結果は保持される

## ロールと権限の詳細

### 組織ロール

| ロール | 説明 | 権限 |
|--------|------|------|
| Owner | 組織のオーナー | 全権限 + 組織削除 + オーナー移譲 |
| Admin | 組織の管理者 | メンバー管理 + 課金管理 + 全プロジェクト Admin |
| Member | 一般メンバー | プロジェクト単位で付与された権限のみ |

### プロジェクトロール

| ロール | 説明 | 権限 |
|--------|------|------|
| Owner | プロジェクトオーナー | 全権限 + 削除・ロール変更不可 |
| Admin | プロジェクト管理者 | プロジェクト設定 + メンバー管理 + 全編集権限 |
| Write | 編集者 | テストスイート/ケースの作成・編集・削除 |
| Read | 閲覧者 | 閲覧のみ（エクスポート可） |

### 権限マトリクス

| 操作 | Owner | Admin | Member(Admin) | Member(Write) | Member(Read) |
|------|:-----:|:-----:|:-------------:|:-------------:|:------------:|
| 組織設定変更 | ○ | ○ | - | - | - |
| メンバー招待 | ○ | ○ | - | - | - |
| 課金管理 | ○ | ○ | - | - | - |
| プロジェクト作成 | ○ | ○ | ○ | - | - |
| プロジェクト削除 | ○ | ○ | ○ | - | - |
| テストスイート作成 | ○ | ○ | ○ | ○ | - |
| テストケース編集 | ○ | ○ | ○ | ○ | - |
| テスト実行 | ○ | ○ | ○ | ○ | - |
| 閲覧 | ○ | ○ | ○ | ○ | ○ |

## 非機能要件

### セキュリティ

| 項目 | 要件 |
|------|------|
| 認証方式 | メール/パスワード + GitHub / Google OAuth 2.0 |
| セッション有効期限 | 7日間（Remember Me 時は 30 日） |
| API トークン形式 | ランダム 32 バイト、SHA-256 ハッシュで保存 |
| 使用量制限 | MCP セッション数は月次でリセット |
| CSRF 対策 | トークンベースの CSRF 保護 |
| XSS 対策 | Content Security Policy 設定 |

### データ保持

| 項目 | 要件 |
|------|------|
| アカウント削除 | リクエストから 30 日後に物理削除 |
| 組織削除 | リクエストから 30 日後に物理削除 |
| 監査ログ保持 | プラン別（Team: 90日、Enterprise: 無制限） |
| 招待有効期限 | 7日間 |
